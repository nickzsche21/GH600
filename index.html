The documentation agent only needs to read files and open pull requests. What is the safest next action?

A Keep admin access and add more logging
B Scope the token to one repository and the two required actions
C Move the admin token into the system prompt
Why B is correct

Least privilege reduces both authorization and blast radius. Logging records misuse; it does not prevent it.
